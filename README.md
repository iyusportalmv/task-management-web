# Task Management Web App

A simple task management web application built to help users manage daily tasks efficiently.

## Features
- Add and delete tasks
- Display task creation date and time
- Persist tasks using browser Local Storage
- Responsive design for desktop and mobile

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Live Demo
Deployment in progress

## Project Purpose
This project was created as a hands-on practice to build a real-world style web application, focusing on DOM manipulation, event handling, and data persistence.

## What I Learned
- Working with DOM and events in JavaScript
- Managing data using Local Storage
- Structuring simple web applications
- Improving usability and accessibility
